Empirical Analysis of Intervening Node Complexity

Testing structural simplicity vs. distance in dependency grammar across 10 languages.

  • Engineered a Python NLP pipeline to parse 2.32M tokens across 102,481 sentences from 10 UD treebanks
  • Extracted structural metrics via recursive subtree computation and arc traversal for UPOS tags and node arity
  • Evaluated 2 formal hypotheses using one-sided paired t-tests, Cohen's d, and pooled UPOS enrichment ratios
  • Linked VERB under-representation (ER = 0.71) to structural load minimization within DLT and DDM frameworks
Result Confirmed 3 hypotheses at p ≤ .001 across 10 languages (Cohen's d 0.23–0.27); 82%+ of interveners had arity ≤ 1, 86%+ had subtree size ≤ 3.

Deep Neural Network Compression & CV

Reducing model size while preserving predictive performance.

  • Built a custom multimodal CNN-MLP architecture from scratch, trained on the Fruits-360 image dataset
  • Engineered a Deep Compression pipeline via magnitude pruning, K-means quantization, and Huffman coding

Multi-Agent Forecasting

A no-regret adaptive ensemble (Hedge + Fixed-Share) for SPY return forecasting, 2010–2024.

  • Four agents — OLS (trend), XGBoost (momentum/volatility), LSTM (sequence) — trained on SPY + VIX features
  • Hedge aggregator with Fixed-Share mixing handles non-stationarity, weights updated online each trading day
  • Backtested across diverse market regimes to minimize drawdowns and maximize risk-adjusted returns
Result Hedge ensemble reached a Sharpe ratio of 0.38 (+58% over a uniform 0.24 baseline); technical agents ~50% directional accuracy.
